# dust (fork)

This is a fork of [dust](https://github.com/dust-tt/dust).

## What this fork adds

This fork adds full-fledged support for environments without an X server,
allowing the project to run headlessly without a graphical display.

## Project status

> [!WARNING]
> **This project is no longer supported.** It is provided as-is, with no
> further updates, maintenance, or support.
